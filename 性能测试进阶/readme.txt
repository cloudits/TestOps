TestOps之性能测试进阶
组织介绍
TestOps是为国内提供测试运维技术服务的组织，其中性能测试进阶课程是专门针对性能测试专项提升的课程体系。
课程介绍
本课程将通过直播模式将实战落实，而基础概念及理论交给录播，这样不但便于学员学习理论基础，并且还有效的降低了学习成本和提高了学习质量。
录播课程体系（超过60个小时的知识点详解，后期更新免费）：
1.网络协议基础与抓包
2.Linux基本命令及简单环境搭建
3.前端开发及调优基础（HTML+CSS+JavaScript）
4.后端开发及调优基础（PHP+JAVA+MySQL+Oracle）
5.LoadRunner工具使用详解
6.Jmeter工具使用详解
7.常见性能测试脚本开发技术（socket、JavaVuser、.net Vuser等）
8.移动端基本开发及调优基础（客户端开发技术、客户端及服务器端性能监控、分析）
9.LoadRunner函数详细介绍（覆盖web及lr所有函数详细分析介绍）
录播课程为知识点模式，每个课程都在5-15分钟，并且提供针对该段课程的知识点列表及常见问题，便于学员查找内容及明确知识点掌握情况。
直播课程体系（6课时*8天）
1.性能测试基本概念及前端快速分析调优思路（yslow、PageSpeed、Dynatrace Ajax……）
2.Linux常见环境搭建、监控命令及运维监控平台部署（ELK、Zabbix、Nmon、Tomcat、redis、Memcache、zookeeper、docker……）
3.基于PHP+Java+Mysql的被测系统开发（自己开发一个系统完成基本数据维护）
4.针对被测系统使用LoadRunner及Jmeter进行性能测试，分析、定位被测系统性能问题
5.性能需求、性能容量规划、性能测试环境规划、性能测试计划、方案、分析调优思路体系（12课时）
6.Jvm+数据库调优实战
7.项目总结，性能测试与持续集成发布，构建自动执行性能测试体系，性能测试报告及简历面试辅导
课程特点
云层多年课程设计及教学经验，针对大多数初学，入门想提高的朋友所制定的课程。没有华丽的词汇，只有踏踏实实的基础、原理、层次推进，真正从实现谈起来让学员站在开发的角度理解被测对象的实现，进一步明了性能瓶颈出现的原理，不是为了让学员“速成”，而是为了让学员明白原理在后期突破的时候“速成”。
录播课程强调实现原理，从前台一步步实现到一步步变慢，再到后台同样类推，真正的去做授之于渔，避免知其一不知其二的尴尬。在直播课程中强调动手实战，通过课前的录播课程让学员有时间学习、预习，而接着直播课程就是将录播内容直接用起来，这样提高了课程的实用性并且也减少了大家学习的疲惫感，当遇到预习不足的知识点可以快速查找相关视频，便于复习。
配合《性能测试进阶指南LoadRunner11》教材，让学员有书看，有知识点查，有视频学，还有实战教，从而做到知其然知其所以然。

在往期的学员中，基本学习时间过半就可以独立进行性能测试工作，并且获得满意的薪水。
